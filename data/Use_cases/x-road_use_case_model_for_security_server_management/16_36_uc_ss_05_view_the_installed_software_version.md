### 3.6 UC SS\_05: View the Installed Software Version

**System**: Security server

**Level**: User task

**Component:** Security server

**Actor**: SS administrator

**Brief Description**: SS administrator views the version of the
installed software.

**Preconditions**: -

administrator.

installed software.
